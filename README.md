Sistema completo de gestão e reserva de quartos, focado na experiência do utilizador e robustez do backend.

##  Stack Tecnológica
* **Backend:** Laravel (PHP)
* **Frontend:** Vue.js & Blade
* **Base de Dados:** MySQL
* **Infraestrutura:** Docker & Docker Compose

##  Funcionalidades
- Listagem dinâmica de quartos.
- Sistema de autenticação de utilizadores.
- Gestão de reservas e disponibilidade em tempo real.
